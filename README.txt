Доброго времени суток!

Основная папка со скриптами - Assets\Scripts.
Скрипты разделены на две папки:
- Physics - основные скрипты, отвечающие за игровую логику;
- Visualization - скрипты Unity, отвечающие за визуализацию и инпут.

Управление:
- Up - ускорение;
- Left/Right - поворот влево/вправо;
- Space - выстрел пушки;
- Left Ctrl - выстрел лазера.
